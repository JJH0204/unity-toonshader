## A. Ramp (LUT) — `_USE_RAMP` / `_RampMap` / `_RampRow`

**실제 동작 (shader 339–341행)**
```hlsl
#if defined(_USE_RAMP)
    half3 ramp = SAMPLE_TEXTURE2D(_RampMap, sampler_RampMap, half2(lightVal, _RampRow)).rgb;
    shadedAlbedo = baseColor.rgb * ramp;   // 음영을 LUT로 계산
#else
    ... 파라메트릭 1·2차 그림자 밴드 (기본) ...
#endif
```
- **정의**: 음영 모델의 **대체 경로**다. 기본은 파라메트릭 1·2차 밴드(결정 #17, 모델러 요구). `_USE_RAMP`를 켜면 대신 **RampMap LUT**로 음영을 계산한다.
- **입력 축**: U(가로) = `lightVal`(0=그림자 … 1=빛, half-Lambert + ILM.g 바이어스 결과). V(세로) = `_RampRow`(부위별 음영 톤 선택용 행).
- **용도**: 손으로 그린 다단계 명암 그라데이션/특수 톤(예: 피부 붉은 경계, 비현실 색조)을 LUT로 직접 저작할 때.
- **모델러 추정과의 차이**: "얼굴이 빛 없는 곳에서 묻히지 않게"는 Ramp가 아니라 **`_ShadeFloor`(최소 밝기 하한)** 와 **`_AmbientStrength`** 가 담당(아래 D 참고). Ramp는 순수 명암 LUT다.
- **기본 상태**: **OFF**. 파라메트릭 밴드가 기본 룩.
- **권장**: 본 프로젝트는 결정 #17로 **파라메트릭 밴드를 표준**으로 한다 → **Ramp는 OFF 유지**(보류). 아트가 LUT 저작 워크플로를 원할 때만 부위별로 ON. (둘은 상호배타: 켜면 밴드 컨트롤은 무시됨.)

## B. ILM 맵 — `_USE_ILM` / `_ILMMap` (RGBA 채널별 실제 의미)

**중립 폴백 (shader 267행)**: ILM off거나 미할당 시 `ilm = (R=0, G=0.5, B=0.5, A=0)`. → 모든 ILM 효과가 무효(중립).

| 채널 | 실제 코드 사용 | 의미 | 중립값 |
|---|---|---|---|
| **R** | `matcapBase = ilm.r` (414, 432행) | **MatCap 마스크 강도**. MatCap/2의 별도 마스크가 off일 때 적용 강도. (PBR 스페큘러엔 영향 없음) | 0 |
| **G** | `shadowBias = (ilm.g - 0.5) * _ShadowOffsetScale` (284행) | **음영 경계 바이어스**. lightVal을 당기거나(>0.5) 밀어(<0.5) 그림자 진입 경계 조절. 밴드·Ramp 양쪽에 적용. | 0.5 |
| **B** | `spec *= (ilm.b * 0.5 + 0.5)` (461행) | **툰 스페큘러 폭 가중**(`_USE_PBR` 시만). 0.5..1로 하이라이트 폭 가중. | 0.5 |
| **A** | `suppressionMask = 1 - ilm.a` (Outline 패스 587행) | **외곽선 억제 마스크**. A=1이면 그 부위 외곽선 제거. | 0 |

**잠긴 규약(CLAUDE.md)과의 실제 차이 (정직한 정의)**
- 규약 "R=스페큘러/MatCap 강도" → **실제는 MatCap 마스크 전용**(일반 스페큘러엔 미적용).
- 규약 "B=스페큘러 폭 / 보조 램프 선택" → **실제는 PBR 스페큘러 폭만**. 보조 램프 선택 기능은 **미구현**.
- 규약 "A=내부선/외곽선 억제" → **실제는 외곽선 억제만**. 내부선(inner-line) 기능은 **미구현**.
- R/G/B/A 패킹 순서·중립값은 규약과 일치.

- **기본 상태**: **OFF**(중립 폴백). 각 효과는 ILM 없이도 자기 컨트롤(MatCap 마스크 슬롯, `_ShadowOffsetScale`, PBR 폭, 외곽선 마스크)로 동작.
- **용도**: 위 4개 마스크를 **한 텍스처에 패킹**해 샘플/메모리 절약. `Window/CharacterToon/ILM Packer`로 분리 마스크 → ILM 생성.
- **권장**: 분리 마스크로 충분하면 **OFF 유지 가능**. 텍스처/샘플 최적화가 필요하거나 ILM 워크플로를 표준화할 때 ON + 패킹. 켤 때 **반드시 ILM 맵 할당**(미할당 시 black 샘플 → R=0/A=0이 중립이라 안전하나 G·B가 0이라 경계/스펙 폭이 한쪽으로 치우침). 단, **Linear로 임포트**(데이터 맵).

## C. 보조 노브 (모델러가 Ramp와 혼동한 "묻힘 방지" 관련)

| 프로퍼티 | 실제 동작 | 용도 |
|---|---|---|
| `_ShadeFloor` (371행) | `shaded = max(shaded, baseColor.rgb * _ShadeFloor)` | **최소 밝기 하한**. 그림자가 완전 검정으로 묻히지 않게(얼굴이 어두운 곳에서 사라지는 것 방지). |
| `_AmbientStrength` (367행) | `ambient = SampleSH(N) * _AmbientStrength * baseColor` | 환경광(SH) 세기. 전체 저조도 보강. |
| `_ShadowOffsetScale` (284행) | ILM.g 바이어스의 스케일 | ILM.g 음영 경계 바이어스의 강도. ILM off면 효과 없음(g=0.5). |
| `_ReceiveShadowStrength` (328행) | `lightVal *= lerp(1, mainLight.shadowAttenuation, _ReceiveShadowStrength)` | 받는 캐스트 그림자 반영도. 운용: 얼굴 0(off)/바디 1(on) — 2-1 참고. |

→ "빛 없는 곳에서 얼굴 묻힘 방지" = **`_ShadeFloor` 올리기 + `_AmbientStrength` 조절**. (Ramp 아님)

## D. 결론 / 사용 방침

- **Ramp**: 정의 = 음영 LUT 대체 경로. 본 프로젝트 표준은 파라메트릭 밴드 → **기본 OFF 유지(보류)**. LUT 저작 시에만 ON.
- **ILM**: 정의 = MatCap마스크(R)/음영바이어스(G)/툰스펙폭(B)/외곽선억제(A) 4-in-1 패킹. **기본 OFF**, 분리 마스크로 충분. 최적화/표준화 시 ON+패킹(Linear).
- **미구현 명시**: ILM의 "보조 램프 선택(B)"·"내부선(A)"은 규약상 예약일 뿐 코드 미구현 → 기대 금지.
- ShaderGUI 툴팁을 위 실제 동작에 맞게 정정함(이 문서와 일치).
