# 09. 합동 시각 검증 체크리스트 (모델러 합동 세션용)

> 2026-06-20 세션에서 구현·Codex 검증·헤드리스 컴파일은 통과했으나 **시각 확인이 미뤄진** 항목 일괄 점검용.
> 진행: ① 사전 준비 → ② 항목별 `확인 방법 → 기대 결과` → ③ 회귀 기준선(깨지면 안 됨).
> 각 항목 끝의 `[ ]`에 합동 세션에서 결과 체크. 문제 시 해당 Docs(06/07/08) 참조.

---

## ① 사전 준비 (USER, 세션 시작 전)

- [ ] **기존 샘플 머티리얼 키워드 재동기화** — 이번에 기본 ON으로 추가된 키워드(`Rim`/`Outline`/`Angel Ring`/`Additional Lights`)는 **키워드를 안 가진 기존 머티리얼에선 OFF로 렌더**된다. 각 머티리얼을 **인스펙터로 한 번 열기**(드로어가 키워드 동기화) 또는 재저장. (Docs/06 2-4 caveat)
- [ ] **URP 렌더러 기능 켜기** — SSAO 효과 확인하려면 PC/Mobile Renderer에 **Screen Space Ambient Occlusion** Renderer Feature 추가. 리플렉션 블렌딩은 씬에 **Reflection Probe 2개+ 겹치게** 배치(+박스 투영 켠 프로브).
- [ ] **URP 그림자 세팅 확인** — `PC_RPAsset`은 Normal Bias=0 + Soft High 적용됨. **`Mobile_RPAsset`도 동일**한지 확인(2-1 #1).
- [ ] (해당 시) **CharacterToonRendererFeature** 렌더러에 추가 + ScreenSpaceOutline 셰이더 지정 — O1/S1(이전 단계) 미완이면.
- [ ] 컴파일 에러 0 (에디터 콘솔) — 이미 확인됨(USER).

## ② 항목별 시각 검증

### P0 — 그림자 / 노말
- [ ] **2-1 그림자 UI 통합**: 인스펙터 Shadow 섹션 하나에 마스크타입/밴드/받는그림자·마스크/Face SDF/Advanced가 모여 있나. **마스크 타입 Enum**(얼굴 mat에서 SDF↔Strength 전환) 동작. **일반 그림자 마스크**(`_ShadowMaskTex`) 칠한 영역에 그림자 억제(눈동자/흰자). 라이트 Y축 회전 시 얼룩 없음(얼굴 Receive OFF / 바디 ON).
- [ ] **2-2 노말맵 근거리 드롭아웃 수정**: 노말맵(Texture Type=Normal map) 할당 후 **카메라를 가까이** 가져가도 디테일 유지(이전엔 근접 시 사라짐). 카메라만 회전→디테일 고정, 라이트만 회전→굴곡 따라 음영 변화.

### P1
- [ ] **2-3 Transparent 모드**: 인스펙터 최상단 Rendering Mode = Transparent → 내부 비침, 툰 셰이딩 유지, 외곽선 블렌드. 부위별 적용(눈은 Opaque). 자기겹침 정렬 아티팩트 정도 확인.
- [ ] **2-4 Use 토글**: Rim/Outline/Normal/ILM/Ramp/MatCap/PBR/Emission/DepthRim/HairShadow/EyeParallax/AngelRing/SSS 각각 on/off 즉시 반영. **MatCap이 마스크/ILM 없이도 표시**(트랩 수정 확인).
- [ ] **2-5 Second MatCap**: 두 번째 MatCap 텍스처 할당 → Add(하이라이트)/Multiply(음영) 블렌드·강도·마스크 동작. (※ URA에 MatCap 스피어 텍스처 없음 → 임시 스피어로 확인)

### P2
- [ ] **3-1 Eye Parallax**: `Use Eye Parallax` ON + A 마스크 든 Eye Overlay 맵 → 가산 오버레이 표시, 카메라 이동 시 시차. (강한 홍채 깊이효과 필요하면 별도 눈 쉐이더 분리 판단 — Docs/06 3-1)
- [ ] **3-2 Stencil 앞머리 투과**: Docs/06 3-2 셋업(헤어=Ref1/Always/Replace, 눈썹/눈=Ref1/Equal/Keep/ZTest Always/높은 큐) → 앞머리 뒤 눈썹/눈 투과. `_StencilMask`로 영역 제어. "모나리자"(눈 사라짐) 해소.

### S2 / R&D / 통합
- [ ] **S2 거리 품질 페이드**: `Use Distance Quality Fade` ON → 카메라 멀리 가면 MatCap/스페큘러/반사/AngelRing/SSS/DepthRim이 자연 페이드(팝 없음), 가까이 복원. Start/End 거리 튜닝. 캐릭터 단위로 페이드(반쪽만 다름 없음).
- [ ] **4-1 PBR UX**: Highlight(라이트 반응) vs Material(메탈릭/반사) 두 하위그룹으로 구분돼 혼동 없는지. (룩 변화 없음 — 정리만)
- [ ] **4-4 SSS(deprecate)**: 기본 OFF. 그림자 밴드 색으로 붉은 산란감 대체 가능한지 → **SSS 하드 삭제 여부 최종 결정**(가치 없으면 Docs/06 4-4 레시피로 제거).
- [ ] **4-5 MatCap 노말 영향도**: `Normal Influence` 낮추면 MatCap의 시점 흔들림(swim) 감소, 0=매끈 클래식. 적정값 결정.
- [ ] **4-6 Angel Ring 방향**: Hair에서 `Angel Ring Angle`로 링을 **가로로 정렬**(기존 세로 문제 해소). `Angel Ring Mask`로 꼭대기/바닥 제외·결 조절.
- [ ] **L3 부가광 셀셰이딩**: 씬에 point/spot 배치 → 캐릭터에 셀 음영 추가, 광원 이동에 반응, 부가광 그림자. `Additional Light Strength` 과발광 시 튜닝. `Use Additional Lights` off 시 키라이트만.
- [ ] **fog**: 씬 Fog 켜고 원거리 → 캐릭터·외곽선이 안개에 자연스럽게 묻힘(분리감 없음).
- [ ] **SSAO**: 렌더러 SSAO 켠 상태 → 크레비스/접합부 음영 깊이. `SSAO Strength`로 과AO(셀 뭉갬) 조절.
- [ ] **리플렉션 프로브 블렌딩**: 프로브 2개 경계 이동 시 PBR 반사가 튀지 않고 블렌딩. 박스 투영 프로브에서 반사 정합.

### 합동 판단 안건 (환경/시각 의존)
- [ ] **4-2 Reflection Strength**: 환경 오브젝트 있는 씬에서 환경 반사 세기 의미 확인 → 유지/조정.
- [ ] **4-3 Outline Depth Offset**: 값 변화로 외곽선 겹침/클리핑 차이 확인 → 유지/제거 결정.
